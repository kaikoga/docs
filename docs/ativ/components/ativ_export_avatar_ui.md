# ATiV Export Avatar UI

<PlatformBadge id="UniVRM0" />
<PlatformBadge id="UniVRM1" />
<PlatformBadge id="DAE" />

このコンポーネントは何もしませんが、Unityのインスペクタに `ndmf:NDMF` または `ablet:Ablet` でアバターをファイルに書き出すためのボタンを表示します。

:::info[INFO]
`ativ:AvatarTinkerVista` バージョン2以下ではVRM専用で、 `Ativ Export VRM0+1 UI` という名前でした。
:::

VRM0.xおよびVRM1.0の書き出しに対応しています。

## 役に立つ状況

NDMFコンソールやAbletコンソールからVRM出力することもできますが、よりお手軽です。

## 使い方

アバター内の任意の場所にコンポーネントを追加してください。
