# Ablet

## 概要

AbletはAvatar Tinker Vistaなどのアバターツールを動かす基盤となるライブラリです。

Abletを使うことで、Unityのアセットに対してプロシージャルな編集および最適化を適用できます。（いわゆる「非破壊改変」です。）

- [主な機能](features.md) : 
- [ウィンドウ](windows.md) : 設定を行うAblet Setting Windowと、ログ出力などを行うAblet Console Windowがあります。
- [プラットフォームサポート](/ablet/supports) : Abletは他の非破壊改変ツールセットと連携して動作します。
- [プラグイン](/ablet/plugins) : Abletはプラグインによって拡張されます。
