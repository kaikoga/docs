# プラットフォームサポート

## 他の非破壊改変ワークフローとの連携

Abletは他の非破壊改変ツールセットと連携して動作します。

- [VRCSDKサポート](vrcsdk.md) : AbletはVRCSDKのビルド時コールバックを利用する他の非破壊改変ツールセットとある程度連携します。
- [NDMFサポート](ndmf.md) : AbletはNDMFと高度に協調して動作します。

## 非破壊改変ワークフローの改善

- [UniVRMサポート](univrm.md) : AbletはUniVRMのVRMエクスポート機能を非破壊改変ワークフローに適した形に修正します。
