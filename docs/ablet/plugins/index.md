# Abletプラグイン

Abletはプラグインによって拡張されます。

## AbletプラグインのNDMF対応

NDMFの存在する環境では、プラグインは以下の３つに分類されます。

- [Ablet専用プラグイン](ablet_plugins.md)
- [NDMF専用プラグイン](ndmf_plugin_support.md)
- [Ablet NDMFハイブリッドプラグイン](hybrid_plugins.md)

:::info[INFO]
`ativ:AvatarTinkerVista` 、 `ew:EmoteWizard` 、 `qr:QuestReplacer_` 、 `arv:AvatarRankerVista` は、いずれもAblet NDMFハイブリッドプラグインです。
:::
