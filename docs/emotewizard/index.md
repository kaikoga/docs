# Emote Wizard

![Emote Wizard](img/emote_wizard.png)

## これは何？

アバターの表情や着せ替えアニメーションのセットアップを支援するツールです。

## 動作環境

Emote Wizardを利用するためには以下の環境が必要です。

- Unity 2022 以上
- VRChatアバターを編集する場合、VRCSDK Avatars 3.5.0以上

`ndmf:NDMF` および `ma:Modular Avatar` を利用する場合に自動的に有効になる追加の機能があります。

- NDMFがインストールされている場合、Emote WizardはNDMFのプラグインとして動作します。
- Emote WizardがNDMFのプラグインとして動作する場合、Generatingフェーズで動作します。
- NDMFを利用した上で、プロジェクトにUniVRMがインポートされている場合、VRM0.xおよびVRM1.0出力をサポートします。

`ablet:Ablet` を利用する場合に自動的に有効になる追加の機能があります。

- Abletがインストールされている場合、Emote WizardはAbletのプラグインとして動作します。
- Emote WizardがAbletのプラグインとして動作する場合、Generatingフェーズで動作します。
- Abletを利用した上で、プロジェクトにUniVRMがインポートされている場合、VRM0.xおよびVRM1.0出力をサポートします。

### 動作確認環境

Emote Wizardは以下の環境で動作を確認しています。

- Unity 2022.3.6f1
- 最新のVRCSDK Avatars
- 最新のAblet
- 最新のUniVRM
